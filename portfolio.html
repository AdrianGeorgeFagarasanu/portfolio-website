Java Chess Engine

A more advanced version of an original chess engine that I had developed that used 2-dimensional arrays and a basic minimax algorithm with alpha-beta pruning.
This edition takes use of the efficiency of keeping concepts closer to machine level code. Instead of using 2D Arrays for the chess board, we can use something called bitboards. Storing 12 different boards, each representing a type of piece in play inside 64-bit 'long'(as opposed to 'int') binary variables, where every bit represents a square on the board. 0 indicated the absence of a piece and 1 represents the presence of one. Specific pieces can be distinguished using bit masking, and all possible moves can be quite efficienctly determined using bitwise operations. This makes additional unique moves quite achievable as well such as castling and en passant.

Status: Work In Progress.

Arduino Lab and Lecture

Developed and presented a 'bootcamp' style lecture/laboratory focused on general circuitry, and Arduino software/hardware. There was a presentation on the required preliminary information followed by a lab activity where students were instructed to make a motion-operated digital countdown timer with a piezo speaker and an array of led lights.
The activity had two main components { a physical circuitry component where schematic wiring was required, and a software component that dealt with the coding in the Arduino Language resembling C }.
